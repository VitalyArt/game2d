# 2D Game Plane (Multiplayer)
###https://game2d.herokuapp.com/
![alt tag](http://oi66.tinypic.com/2hhd5l2.jpg)

Install game
```sh
git clone https://github.com/VitalyArt/game2d.git
cd game2d
npm install
```
Open app.js and change the line
```js
var PORT = 80;
```
Run in Terminal
```sh
node app.js
```
